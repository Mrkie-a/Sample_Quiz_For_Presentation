## 📌 Project Overview

The **Internet History Quiz System** is a web-based quiz application designed for educational purposes.  
It includes a **20-item multiple choice exam**, timer system, and score calculation using PHP and MySQL.

## ⚙️ How It Works

1. Student enters name & course  
2. Quiz starts with timer ⏱  
3. Answer 20 questions  
4. System calculates score  
5. Results saved in database

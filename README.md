## ✨ Overview
A **web-based quiz system** that tests knowledge about the **History of the Internet**.  
Built for school presentation and learning web development fundamentals.

---

## 🚀 Features

✔ 20 Multiple Choice Questions  
✔ A, B, C, D styled options  
✔ Countdown Timer ⏱  
✔ Student session (Name & Course)  
✔ Score calculation system  
✔ MySQL database integration  
✔ Clean and responsive UI  

---

## 🧠 Quiz Topics

- Origins of the Internet (ARPANET)
- Key protocols (TCP/IP, FTP, HTTP)
- Web technologies (HTML, CSS)
- Internet communication tools
- Domain systems (.com, .edu, .gov)
